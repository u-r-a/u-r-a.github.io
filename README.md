# u-r-a.github.io

Personal site hosting project posters and resume. Served via GitHub Pages at https://u-r-a.github.io

## Add content
- **Resume:** drop a `resume.pdf` in the repo root.
- **Posters:** drop PDFs into `posters/`, then add a matching `<li>` link in `index.html`.

## Publish updates
```sh
git add . && git commit -m "Update site" && git push
```
Pages redeploys automatically within ~1 minute.
